无 Python 2017/07/19

除法

  1. X / Y 2.6之前的版本会省去小数部分，3版本之后无论任何类型都会保持小数部分。
  2. X // Y Floor除法，不管哪个版本，总是省略小数部分,也称为截断除法，结果会向下舍入
    import math
    print(math.floor(-2.5)) //输出-3 （向下舍入）
    print(math.floor(2.5)) //输出2 （向下舍入）
    print(math.trunc(5 / -2))//输出-2
    print(math.trunc(5 / 2))//输出2

如果你真的只想截断，而不管符号，可以通过math.trunc来得到一个浮点除法
